# 📊 Büyüme ve Ölçekleme

## 🔢 Rakamlar

Zirve döneminde **154 aktif müşterimiz** vardı, aylık ciromuz yaklaşık **20.000₺** civarındaydı. Toplamda bizden **203 kişi** satın alım yaptı.

Bugüne kadarki gelirleri çıkarttığımızda, bu iş üzerinden toplamda **250.000-300.000₺** civarında zararımız var gibi görünüyor.
